Example 1
---------
| Description: Uniformly magnetized spherical nanoparticle benchmark.
This example provides the 3D structural and magnetic configuration of a homogeneous spherical nanoparticle together with the corresponding NuMagSANS input and output files. It is intended as a simple benchmark and validation case for testing nuclear and magnetic SANS calculations against a well-defined reference geometry.

Example 2
---------

| Description: Linearized vortex benchmark in a spherical nanoparticle. 
A reference dataset for a spherical nanoparticle with a linearized vortex magnetization state, designed for benchmarking and validation of the NuMagSANS framework. The record includes real-space nuclear and magnetic input data, a data-generation script, a NuMagSANS configuration file, and the corresponding simulated SANS cross sections.
|

Example 3
---------
| Description: Dilute ensemble of spherical iron nanoparticles with cubic anisotropy.
A reference workflow for simulating a dilute ensemble of spherical iron nanoparticles with cubic magnetocrystalline anisotropy using MuMax3 and NuMagSANS. The dataset includes the full simulation and post-processing pipeline, covering random-field hysteresis simulations, back-rotation into a common reference frame, and the corresponding magnetic SANS cross sections for benchmarking and validation.
|
| Zenodo Link: https://doi.org/10.5281/zenodo.17931156
| Reference: M. P. Adams, A. Michels, arXiv:2601.18444
| Preprint: https://doi.org/10.48550/arXiv.2601.18444
