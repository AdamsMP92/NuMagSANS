* Tutorial Goal
- Introduce the NuMagSANS workflow from real-space input data to SANS output.
- Show how a minimal simulation directory is organized.
- Keep this file as a reusable slide template for future NuMagSANS tutorials.

* NuMagSANS Workflow
- Prepare real-space input data.
- Configure the NuMagSANS input file.
- Run the CUDA backend through the Python facade.
- Analyze CSV or HDF5 output data.

* Real-Space Data
- ~MagData~ stores local magnetic moment data.
- ~NucData~ stores local nuclear scattering length density data.
- ~StructData~ stores object positions.
- ~RotData~ stores object rotations.

* Output Backends
- CSV is the default output format.
- HDF5 can be enabled at build time for compact structured output.
- Both output modes follow the same logical data hierarchy.
