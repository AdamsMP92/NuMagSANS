- Data organization for ensemble simulations.
- StructData and RotData loops.

* Simulation Layers
- Local object data describe one or more materialized objects.
- Structure data places these objects in global space.
- Rotation data orients these objects in global space.
- Output data can be exported as CSV or HDF5.
